# Nici's Crib

Lokale PWA fur Wohnungs-, Umzugs- und Haushaltsausgaben.

## Entwicklung

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Die fertige statische App liegt danach in `dist`.

## GitHub Pages

Dieses Repository enthaelt einen GitHub-Actions-Workflow unter `.github/workflows/deploy.yml`.
Nach dem Push auf `main` baut GitHub die App automatisch und veroeffentlicht sie mit GitHub Pages.
